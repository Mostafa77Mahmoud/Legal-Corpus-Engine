فيه حاجات اتحسنت فعلًا، لكن لسه في ملاحظات مهمة قبل ما تعتبر الـ pipeline "مطابق للخطة".

## 1. `target_article_ids` ممتاز

الوكيل نفذ حاجة كنت شايفها مهمة:

بدل:

```json
{
  "article_numbers": [4]
}
```

بقى:

```json
{
  "article_numbers": [4],
  "target_article_ids": ["EG_ESIGN_004"]
}
```

وده هيفيد جدًا في الـ retrieval والـ graph navigation.

---

## 2. `concepts: []` مقلق

مثال المادة 5:

```json
"concepts": []
```

في قانون التوقيع الإلكتروني.

ده مش طبيعي أوي.

مادة بتتكلم عن:

* الهيئة
* التراخيص
* الرسوم
* الإيرادات

المفروض غالبًا يبقى فيها concepts زي:

```json
administrative_authority
licensing
fees
regulatory_compliance
```

لو معظم المواد طالعة:

```json
[]
```

يبقى عندك واحدة من 3 مشاكل:

### احتمال 1

taxonomy ضيقة جدًا

### احتمال 2

Gemini بيرجع concepts مختلفة

مثلاً:

```json
authority
license
fees
```

وأنت عامل validation صارم فبيترفضوا كلهم.

### احتمال 3

Prompt ضعيف.

---

### اطلب منه:

```text
Audit concepts extraction.

Generate a report showing:

1. Total articles
2. Articles with at least one concept
3. Articles with zero concepts
4. Top 50 concepts returned by Gemini before taxonomy filtering
5. Top 50 concepts after taxonomy filtering

Determine whether low concept coverage is caused by:

- Prompt quality
- Taxonomy mismatch
- Validation logic

Do not modify code yet.
Produce diagnostics only.
```

## 3. `applicable_to` محتاج مراجعة

المادة دي طلعلها:

```json
[
  "administrative",
  "commercial",
  "digital"
]
```

ممكن تكون صح.

لكن لو لقيت 90% من مواد القانون:

```json
administrative
commercial
digital
```

يبقى الموديل بيرمي نفس التلات قيم لكل حاجة.

وده هيخلي الحقل عديم القيمة.

---

## 4. الـ OCR Fix اليدوي مش عاجبني

هو عمل:

```text
هيية -> هيئة
رييس -> رئيس
```

في ملفات split_articles مباشرة.

ده حل إسعافي.

لكن مش حل هندسي.

المفروض السبب الأصلي يكون متصلح.

لو بكرة ظهر:

```text
مسيول
شيون
مسيولية
```

هتفضل تطارد الكلمات واحدة واحدة.
