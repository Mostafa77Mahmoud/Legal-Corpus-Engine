# V7 Minimal Production Architecture



## Stage 1 — Extraction



### الهدف



استخراج النص من PDF بأعلى دقة وأقل تكلفة.



### الآلية



```text

PDF

 │

 ▼

PyMuPDF Extraction

 │

 ▼

Confidence Check

 │

 ├─ Confidence >= 0.85

 │      └─ Accept

 │

 └─ Confidence < 0.85

        └─ Gemini OCR Fallback

```



### المخرجات



```text

data/extracted_raw/

```



مع:



```json

{

  "extraction_source": "pymupdf"

}

```



أو



```json

{

  "extraction_source": "gemini_ocr"

}

```



---



## Stage 1.3 — Cleanup



### الهدف



تنظيف النص فقط.



بدون LLM.



### أمثلة



* إزالة أرقام الصفحات

* إزالة Watermarks

* توحيد الحروف العربية

* إزالة المسافات المكسورة



### المخرجات



```text

data/extracted_clean/

```



### مطلوب



Audit Log



```text

data/cleanup_audit_logs/

```



يسجل كل تعديل تم.



---



## Stage 1.5 — Confidence Scoring



### حساب



```text

AMD

ACD

EACC

SHC

CS

```



ثم:



```text

Confidence =

(AMD × 0.25)

+

(ACD × 0.25)

+

(EACC × 0.25)

+

(SHC × 0.15)

+

((1-CS) × 0.10)

```



كل Metric لازم تكون normalized بين:



```text

0 → 1

```



### إذا



```text

confidence < 0.85

```



يتم:



```json

{

  "manual_review": true

}

```

# Stage 3.7 — Chunking



### مهم جداً



اتفقنا على Paragraph First Chunking



وليس Fixed Split.



### الطريقة



```text

Paragraph

 ↓

Paragraph

 ↓

Paragraph

```

```text

≈ 400 token

```



مع حد أقصى:



```text

500 token

```



لو فقرة أكبر من 500:



```text

Split on sentence boundary

```



وليس منتصف الجملة.
